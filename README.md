
# RGB LED Control with HTTP web server

RGB LED Conterol with HTTP web server in ESP-IDF


## Build

``` 
  idf.py set-target esp32s3
  idf.py menuconfig
```
change your ssid and password in Example Configuration then build and flash.

```
 idf.py build
```
